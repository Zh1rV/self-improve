# Preflight Checklist

Run this checklist before the first unattended iteration.

- Confirm current repository path and active branch.
- Confirm working tree status (clean or known dirty state).
- Confirm available validation commands can be executed locally.
- Confirm no pending risky operations requiring user approval.
- Confirm iteration budget and risk profile are set.
- Confirm rollback path exists (typically git-based revert/reset strategy).
